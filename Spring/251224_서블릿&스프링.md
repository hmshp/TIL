## 서블릿 컨테이너

- 인터넷 환경(웹)에서 프로그램이 동작하려면 HTTP 프로토콜로 된 request - response 통신이 필요한데, 이런 통신 기능을 제공하는 것이 자바의 Servlet API다.
<img width="730" height="402" alt="image" src="https://github.com/user-attachments/assets/86d483c6-da7f-40f4-bb13-dbb2e00c5dc0" />


- 그래서 HTTP 요청을 처리해 주는 컴포넌트들을 Servlet이라고 부른다.
- Servlet은 하나가 아니고 여러 개고, 여러 Servlet들을 관리해 주는 게 서블릿 컨테이너다(대표적으로는 Tomcat).
- 서블릿 컨테이너는 HTTP 요청과 관련된 작업(ex) 맵핑), 서블릿이 동작할 수 있게 메모리에 올리는 등의 작업을 해서 실행시켜 주고 컴포넌트 라이프사이클 관리도 해 준다.
